Local | Session | Cookie ===>>> Storage(s)

  1. Local Storage

    • 10 mb
    • Never expires - Некогда не истекает
    • Manually configured - Настраевается вручную

    Ruleset - Команды

    • GET -> localStorage.getItem('key')
      Example:
      - const usersJSON = localStorage.getItem('users')
      - const usersArr = JSON.parse(usersJSON)
    • SET -> localStorage.setItem('key', JSON.stringify(value))
      Example:
      - const users = [{name: 'Andrew', age: 27}, {name: 'Vikram', age: 31}]
      - localStorage.setItem('users', JSON.stringify(users))
    • REMOVE -> localStorage.removeItem('key')
      Example:
      - localStorage.removeItem('users')
    • CLEAR -> localStorage.clear()
      Example:
      - localStorage.clear()

  2. Session Storage

    • 5 mb
    • Gets deleted when the tab is closed!
    • Can be manually configured too

    Ruleset - Команды

    • GET -> sessionStorage.getItem('key')
      Example:
      - const usersJSON = sessionStorage.getItem('users')
      - const usersArr = JSON.parse(usersJSON)
    • SET -> sessionStorage.setItem('key', JSON.stringify(value))
      Example:
      - const users = [{name: 'Andrew', age: 27}, {name: 'Vikram', age: 31}]
      - sessionStorage.setItem('users', JSON.stringify(users))
    • REMOVE -> sessionStorage.removeItem('key')
      Example:
      - sessionStorage.removeItem('users')
    • CLEAR -> sessionStorage.clear()
      Example:
      - sessionStorage.clear()

  3. Cookie

    • 4 kb
    • Never gets deleted until expiration date comes
    • Expiration date is set manually
